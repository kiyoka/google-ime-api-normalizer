# Google IME API Normalizerとは

* Google CGI API for Japanese Input  http://www.google.com/intl/ja/ime/cgiapi.html のプロキシです。
* 同APIの返却するJSONがRFC4627に準拠していないのを修正して返します。
* heroku.comでデプロイできる予定です。

## Requires
 - Ruby 1.9.1
 - rack
 - sinatra
